How to Prepare Your Artwork for Custom Printing: File Format Guide

📅 March 2026 🕒 8 min read ✍️ June's Tees & Things

The most common reason a print order gets delayed or comes out wrong is not the printing — it is the artwork file. Understanding a few key concepts about print files will save you time, money, and frustration on every order you ever place.

Vector vs. Raster: The Most Important Concept in Print Artwork

Every file format you have ever worked with falls into one of two categories: vector or raster. Understanding the difference between them is the single most important thing a customer can know before submitting artwork to a print shop.

Raster images are made of pixels — tiny squares of color arranged in a grid. JPG, PNG, GIF, BMP, TIFF, and PSD are all raster formats. The problem with raster is that they have a fixed resolution. When you stretch a raster image larger than its native size, you are just making the pixels bigger — which creates blurriness, jagged edges, and visible "stairstepping" on curves and diagonals. A logo saved as a 200 × 200 pixel JPG looks fine as a thumbnail on a website, but printed at 4 inches wide on a shirt it will be a blurry mess.

Vector images are made of mathematical paths — lines, curves, and shapes described by coordinates and equations. AI (Adobe Illustrator), EPS, SVG, and PDF (with embedded vector data) are vector formats. Because vectors are mathematical, they can scale to any size — a postage stamp or a billboard — and always output at perfect sharpness. One vector logo file works for a business card and a semi-truck wrap.

If you have a logo designed for you, always request the vector source file (AI or EPS) in addition to any JPG or PNG previews. The vector file is the master file that every future print job should use.

Adobe's Creative Cloud design resource hub covers vector design fundamentals, file formats, and print preparation in depth.

The Golden Rule

If you can zoom into your logo file until it fills the screen and it stays perfectly sharp with clean edges, it is a vector file. If it gets blurry or pixelated, it is a raster file that needs to be redrawn as a vector before it can be printed cleanly at large size.

Why That Logo From the Internet Will Not Work

We see this every week: a customer sends a logo they right-clicked off a company website or screenshot from Instagram. Web images are optimized for small file size and screen display — typically 72–96 DPI and 200–500 pixels wide. This is completely unusable for print.

The minimum resolution for high-quality print at actual print size is 300 DPI. A 4-inch wide design printed at 300 DPI requires a 1200 × 1200 pixel file minimum. A 10-inch wide design requires 3000 pixels. The image you grabbed from the website is not even close.

There is no software that can turn a low-resolution raster image into a high-quality print file. AI upscalers can sharpen photos somewhat, but they cannot invent detail that was never there. The only real solution is to either locate the original high-resolution source file or have the logo redrawn as a vector by a designer.

CMYK vs. RGB: Color Mode Matters

Computer screens create color by mixing red, green, and blue light (RGB). Printers create color by layering cyan, magenta, yellow, and black ink (CMYK). These two systems have different color gamuts — the range of colors they can reproduce — and they do not map to each other perfectly.

The practical implication: if you design a logo in RGB with a vivid electric blue (#0057FF) and submit it for screen printing, the printed result will look noticeably different — typically darker and less saturated — because CMYK cannot reproduce some RGB colors.

Always design print-bound artwork in CMYK mode from the start. In Adobe Illustrator or Photoshop, set your color mode to CMYK before beginning. If you are using a tool that only supports RGB (like most web-based design tools), the conversion at submission time can be managed, but flag it to your printer so they can adjust if needed.

Canva is a popular design tool but operates entirely in RGB and has limited support for print-accurate color output. It is a useful starting point for concepts but has real limitations for production files. Canva's own design tips library acknowledges these print limitations and recommends working with a professional printer's specifications.

The 300 DPI Rule

300 DPI (dots per inch) is the standard minimum resolution for printed materials viewed at normal reading distance — shirts, business cards, flyers, brochures. Large format items (banners, signs, backdrops) can use 100–150 DPI because they are viewed from farther away. Magazine-quality print uses 300–400 DPI.

When setting up a file for screen printing, your artwork should be 300 DPI at actual print size. If your design will print at 12 inches wide, the file should be at least 3600 pixels wide. If it will print at 4 inches wide, 1200 pixels wide is sufficient.

Outline Your Fonts

Fonts are separate software components — your design file references a font installed on your computer, but that font may not be installed on your printer's production computer. If you send an AI file with live text and the printer does not have your font, the text will either substitute a default font (completely changing your design) or show as an error.

The solution: outline (or expand) all fonts before submitting. In Adobe Illustrator, select all text and go to Type → Create Outlines. This converts your text from live type into vector shapes — the printer's software has no font dependency, the shapes are embedded in the file, and your design looks exactly as you intended. Do this on a copy of your file and keep the original with live text for future editing.

Pantone and Color Matching

For brand-critical colors — corporate logos, school colors, team colors — Pantone matching is the professional standard. The Pantone Matching System (PMS) assigns a number to every standardized ink color. PMS 286 is a specific shade of blue. PMS 165 is a specific orange. Every professional print shop can mix ink to a Pantone number with high accuracy.

If you do not have a Pantone reference, provide a CMYK value or a hex code and we will get as close as possible. For screen printing, we can pull a physical Pantone swatch and mix ink to match. For DTF and digital printing, CMYK is the reference system.

What "Print-Ready" Actually Means

When a printer asks for a "print-ready" file, they mean a file that requires no additional preparation before it goes on press. A print-ready file:

  • Is vector or 300 DPI raster at actual print size
  • Is in CMYK color mode (or has CMYK values specified)
  • Has all fonts outlined or embedded
  • Has correct bleed and safe zone margins (for banners and signs)
  • Has no RGB or spot colors that have not been converted or specified
  • Is saved in a production format: PDF, AI, EPS, or high-resolution TIFF/PNG

When to DIY vs. When to Hire a Designer

Tools like Canva, Adobe Express, and Figma have made basic design accessible to everyone, and many customers produce perfectly usable artwork for simple jobs. If you have a clear vision and time to learn the tool, DIY is a reasonable approach for text-based designs, simple layouts, and projects where exact color matching is not critical.

Hire a designer when: you have a brand that needs consistency across multiple materials, your logo does not yet exist as a vector file, you need a photorealistic or complex illustration, or the stakes are high enough that bad design would cost you more than design fees. 99designs' design blog has excellent guides on when professional design is worth the investment and how to brief a designer effectively.
